High-Precision Mobile Mapping & Localization System

Integrated JAWESO Panora (360° multi-camera) and Planar systems into a mobile mapping pipeline with RTK-GNSS, LiDAR, and IMU fusion — achieving <10 cm absolute positional accuracy for digital twin generation and infrastructure inspection.

Sensor Fusion Localization LiDAR + Visual Odometry ROS 2 RTK-GNSS IMU Multi-Camera Calibration Point Clouds Digital Twins Python
Problem

Centimetre-accurate mapping at scale

Urban and railway infrastructure inspection demands georeferenced 3D reconstruction with sub-decimetre accuracy from a moving vehicle. This requires fusing heterogeneous sensors (6-camera panoramic rig, LiDAR, IMU, RTK-GNSS, wheel odometry) with tight time synchronisation and robust calibration across wide-angle imaging conditions.

Architecture

End-to-end pipeline

1 — Sensor Synchronisation
Designed time-alignment between vehicle odometry, NMEA/PPS triggers, and multi-camera capture. Ensured all modalities share a consistent timebase for downstream fusion.
NMEA/PPS trigger → hardware sync → unified timestamp
2 — Calibration
Multi-camera intrinsic and extrinsic calibration for panoramic and planar rigs. Camera–LiDAR–IMU extrinsic calibration with sub-pixel reprojection accuracy, robust across wide-angle imaging conditions.
Intrinsic cal → cross-modal extrinsic cal → reprojection validation
3 — Multi-Sensor Fusion & Localization
Fused RTK-GNSS, LiDAR, IMU, and camera data for vehicle localization. Developed both visual and LiDAR-based odometry pipelines, achieving consistent <10 cm absolute positional accuracy.
RTK + LiDAR + IMU + VO → fused pose estimation
4 — Georeferenced Point Cloud Generation
Generated dense, georeferenced point clouds from panoramic and planar imagery combined with LiDAR geometry. Output suitable for digital twin and inspection workflows.
Fused pose + LiDAR + imagery → georeferenced 3D map
5 — Automated Data Pipeline
End-to-end automation: data ingest → mapping → localization → visualisation. Implemented in ROS 2 with custom Python tooling. Plug-and-play integration into the existing localization stack.
ROS 2 orchestration → automated ingest → export

Project 05 • Healthcare Robotics • End-to-End System • ROS 2

Healthcare Robotics System

Full end-to-end design and implementation of a nursing-assist robot — covering mobile base control, autonomous navigation, dexterous manipulation with a 7-DOF arm + dexterous hand, teleoperation, and a mission-control UI. Built entirely on ROS 2 with real-time hardware interfaces.

ROS 2 Control Autonomous Navigation Manipulation xArm7 Inspire Hand Nav2 MoveIt 2 micro-ROS / DDS Leap Motion Force Sensing Qt UI URDF / SDF
Problem

A complete robotics platform from scratch

Healthcare environments need robots that can navigate autonomously indoors, manipulate objects dexterously, and be teleoperated by clinical staff — all while meeting real-time control requirements and safety constraints. This project required designing every layer: from motor-level hardware interfaces to high-level mission planning.

Subsystem 1

Mobile base — hoverboard motors + ros2_control

Hardware Interface ros2_control
Designed a real-time hardware interface for hoverboard motors using ros2_control. Wrote custom hardware interface plugins to bridge motor drivers with the ROS 2 control framework, achieving deterministic update rates for differential drive.
Microcontroller Integration micro-ROS / DDS
Integrated embedded motor controllers with the ROS 2 graph via micro-ROS (micro-DDS agent), enabling real-time command/feedback loops between the microcontroller and the main compute.
Subsystem 2

Autonomous indoor navigation

Perception
LiDAR and camera-based perception for indoor mapping and obstacle detection. Sensor data fused for robust localisation in cluttered hospital-like environments.
LiDAR + cameras → costmap → localisation
Planning & Execution Nav2
Full Nav2 stack for path planning, behaviour trees, and recovery actions. Autonomous point-to-point navigation in indoor environments with dynamic obstacle avoidance.
AMCL localisation → Nav2 planner → controller → recovery
Subsystem 3

Dexterous manipulation — xArm7 + Inspire Hand

Robot Description & Modelling
Wrote URDF/SDF for the full kinematic chain: xArm7 + Inspire dexterous hand. Added collision meshes, joint limits, and transmission definitions. Integrated with xArm7's RT serial module for real-time joint-level control.
URDF/SDF authoring → collision geometry → ros2_control integration
MoveIt 2 Configuration
Full MoveIt 2 setup for xArm7 + Inspire Hand: planning groups, IK solvers, collision detection, and self-collision avoidance. Configured MoveIt Servo for real-time teleoperation input. Planning to upgrade to NVIDIA cuMotion for GPU-accelerated motion planning.
MoveIt 2 config → IK/FK → collision checking → Servo mode
Force Sensing
Integrated force/torque sensor for contact-aware manipulation. Enables compliant grasping and safe interaction in healthcare tasks.
F/T sensor → contact detection → compliant control
Subsystem 4

Teleoperation — Leap Motion + MoveIt Servo

Hand Tracking → Robot Control
Integrated Leap Motion sensor with MoveIt 2 Servo for intuitive teleoperation of the xArm7 + Inspire Hand. Operator hand poses are mapped in real-time to robot end-effector commands, enabling natural dexterous control for remote tasks.
Leap Motion → pose mapping → MoveIt Servo → real-time joint commands
Subsystem 5

Mission control — Qt UI

Nursing-Assist Mission Interface Qt / ROS 2
Built a Qt-based operator UI for defining and executing nursing-assist missions: room-to-room delivery, patient check-in workflows, and status monitoring. Communicates with Nav2 and manipulation stacks via ROS 2 actions/services.
Qt UI → ROS 2 action client → Nav2 + MoveIt 2

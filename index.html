About

Autonomy software, end to end

I'm a Robotics Software Engineer who architects real-time perception and navigation systems in modern C++ — from sensor drivers and multi-threaded data pipelines to graph-based SLAM and the estimators that fuse LiDAR, cameras, IMU, and RTK-GNSS into one consistent state.

My core is state estimation and optimization: LiDAR-inertial odometry, factor-graph optimization for localization and mapping, and tightly-coupled sensor fusion (EKF/UKF). I treat the numerics and the systems as one problem — deterministic real-time behaviour, concurrency, and performance budgets that survive on embedded targets like NVIDIA Jetson.

I build for the field, not the bench. My stacks run on industrial platforms with RTK-grade sensors, GPS-denied inspection drones flying on PX4, and outdoor mobile robots — where glare, signal denial, and real-world tolerances decide whether the system holds. Across full-time and contract roles I've shipped production autonomy end to end, from sensor integration to the navigation running on the robot.

Overview

Automating a precise, repeatable clinical task

Dental cleaning is precise, repetitive, and physically demanding — a strong fit for a robot that can hold an exact tool path and repeat it reliably. I built a working prototype that automates the routine cleaning workflow while keeping the interaction safe and compliant against a real dental surface.

What I Built

Perception, manipulation, and integration

  • 3D vision system — stereo/depth cameras to localize the work area and register the tool to the surface, driving the arm in real time.
  • 7-DOF arm guidance — real-time motion control turning vision targets into precise, repeatable tool trajectories.
  • Position & contact sensing — continuous feedback for safe, compliant tool interaction — the tool follows the surface instead of pushing through it.
  • Dental tooling end-effector — integrated specialized dental tooling on the end-effector for the actual cleaning operation.
  • Clinical usability & hygiene — engineered the system around hygiene and clinical-usability requirements.
Result

A functioning prototype

The outcome is a working prototype demonstrating controlled, vision-guided cleaning operations — an end-to-end proof that perception, manipulation, and safe contact control can be combined into a usable clinical tool.
